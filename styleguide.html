Navigation Button

The navigation button is part of the navigation menu bar.

CSS selector:

.button-nav

Sample code:

<a href="#" class="button">NA (89 - 97)</a>

Rendered element:

NA (89 - 97)

In-text link

CSS selector:

.in-text-link

Sample code:

<a class="in-text-link" href="#">In-text Link Sample</a>

Rendered element:

In-text Link Sample

Submit/Reset Buttons

CSS selector:

.button-reset, .button-submit

Sample Code:

<a class="button-reset" href="#">Reset</a>
<a class="button-submit" href="#">Submit</a>

Rendered Element:

RESET SUBMIT

Bulleted Lists

CSS selector:

.bulleted-list

Sample code:

<ul class="bulleted-list">
<li>NA (89-97)</li>
 <li>NB (98-05)</li>
 <li>NC (06-14)</li>
 <li>NC (15+)</li>
 </ul>

Rendered element:

  • NA (89-97)
  • NB (98-05)
  • NC (06-14)
  • ND (15+)

Numbered Lists
