.link

Bulleted Lists

CSS selector:

.list-bullet

Sample code:

<ul class="list-bullet"> <li>A line of some text</li> <li>Another line of more text</li> </ul>

Rendered element:

  • A line of some text
  • Another line of more text

Numbered Lists

CSS selector:

.list-number

Sample code:

<ol class="list-number"> <li>A line of some text</li> <li>Another line of more text</li> </ol>

Rendered element:

  1. A line of some text
  2. Another line of more text
